# Rediseño y Optimización del Sitio Web del Colegio Juan de Dios Carvajal

## Descripción del Proyecto
Vamos a reconstruir y modernizar la página web del Colegio Juan de Dios Carvajal ([https://iejuandedioscarvajal.edu.co/](https://iejuandedioscarvajal.edu.co/)), creando una plataforma digital que sea funcional, atractiva y fácil de usar para todos los miembros de la comunidad educativa (estudiantes, padres, profesores y personal administrativo). El nuevo sitio web será una herramienta centralizada para acceder a información, recursos y servicios del colegio.

## Justificación
La actual página del colegio está mal estructurada, no tiene óptimas condiciones, contiene muchas páginas innecesarias y está muy desordenada. Este proyecto busca solucionar estos problemas proporcionando una experiencia de usuario mejorada y una estructura más clara y eficiente.

## Organización del Equipo
El equipo está compuesto por 4 integrantes, cada uno con un área específica de responsabilidad:

1. **Coordinador del Proyecto**:
   - Supervisa el progreso general del proyecto.
   - Asegura que se cumplan los plazos y que el equipo esté alineado.

2. **Diseñador UI/UX**:
   - Encargado del diseño de la interfaz de usuario y la experiencia de usuario.
   - Crea wireframes, prototipos y se asegura de que el diseño sea atractivo y funcional.
   - Trabaja en la parte visual (colores, tipografías, iconos, etc.).

3. **Desarrollador Frontend**:
   - Responsable de implementar el diseño en código (HTML, CSS, JavaScript).
   - Asegura que el sitio sea responsive y compatible con diferentes navegadores.
   - Trabaja en la interactividad y la usabilidad del sitio.

4. **Desarrollador Backend**:
   - Encargado de la lógica del servidor y la base de datos.
   - Implementa funcionalidades como el CMS, formularios de contacto, o integración con APIs.
   - Asegura que el sitio sea seguro y escalable.

## Herramientas de Colaboración

## Presentación Final
La presentación final del proyecto se realizará mediante una exposición donde se explicará el proceso de desarrollo del proyecto:

### Desarrollo del Proyecto (10-15 minutos)
1. **Investigación y Planificación**:
   - Explicar cómo se identificaron las necesidades del colegio y los usuarios.
   - Mostrar el plan de trabajo y la metodología utilizada.

2. **Diseño**:
   - Presentar los wireframes y prototipos iniciales.
   - Explicar las decisiones de diseño (colores, tipografías, estructura, etc.).

3. **Desarrollo**:
   - Mostrar el proceso de desarrollo frontend y backend.
   - Destacar las tecnologías utilizadas (HTML, CSS, JavaScript, etc.).

4. **Funcionalidades Clave**:
   - Demostrar las funcionalidades principales del sitio web (calendario académico, noticias, formularios de contacto, etc.).
   - Explicar cómo estas funcionalidades benefician a la comunidad educativa.

### Resultados y Demostración (10 minutos)
1. **Demo en Vivo**:
   - Navegar por el sitio web en vivo, mostrando las diferentes secciones y funcionalidades.

2. **Comparación (Antes y Después)**:
   - Mostrar capturas de pantalla o una breve comparación entre la página web antigua y la nueva.

3. **Beneficios del Proyecto**:
   - Resaltar cómo el nuevo sitio web mejora la experiencia del usuario y cumple con los objetivos planteados.

### Conclusión (5 minutos)
1. **Lecciones Aprendidas**:
   - Compartir los desafíos enfrentados y cómo se resolvieron.
   - Mencionar lo que el equipo aprendió durante el proyecto.

2. **Próximos Pasos**:
   - Hablar sobre el mantenimiento y posibles mejoras futuras.

3. **Agradecimientos**:
   - Agradecer a los profesores, stakeholders y cualquier persona que haya apoyado el proyecto.
